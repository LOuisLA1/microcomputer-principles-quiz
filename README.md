# 微机原理期末复习测验

单文件 HTML 在线刷题工具，基于课堂测验及核心题目精选题库，覆盖 8086 微机原理期末考试七大模块。

## 内容模块

| 模块 | 题数 |
|------|------|
| 微处理器（8086 架构、寄存器、标志位） | 10 |
| 寻址方式和传送类指令 | 10 |
| 常用指令与逻辑运算 | 10 |
| 输入输出接口 | 8 |
| 存储器基础知识 | 10 |
| 中断系统 | 10 |
| 8253 定时计数器 | 10 |

## 功能

- 按模块筛选，随机打乱，计时
- 即时判对错，每题附带详细解析
- 错题本（localStorage 持久保存）
- 深色/浅色模式切换
- 键盘操作：`1`-`4` 选答案，`←` `→` 翻题
- 零依赖，双击 HTML 文件即可使用

## 使用方式

**在线版**：https://louisla1.github.io/microcomputer-principles-quiz/

本地使用：下载 `微机原理学习通题目.html`，浏览器打开即可。
